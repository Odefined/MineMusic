# Music Intelligence

Music Intelligence owns read-oriented retrieval and knowledge capabilities for
agent reasoning support. It does not own durable music facts, long-term Memory,
presentation cards, external effects, or final recommendation judgement.

Current area docs:

- `design.md`: Music Intelligence Retrieval boundary and non-goals.
- `ports.md`: provided/consumed ports, dependency rules, and guards.
- `progress.md`: current implementation state and verification notes.
