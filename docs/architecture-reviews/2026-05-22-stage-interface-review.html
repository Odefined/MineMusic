Keep host adapters thin

Worth exploring ports & adapters

After

flowchart TB

  CLI[CLI adapter] --> StageInterface
  WebUI[WebUI adapter] --> StageInterface
  StageInterface --> Runtime[Runtime composition]

Problem

The MCP module knows tool shape and default runtime shape, so future hosts may repeat knowledge.

Solution

Host adapters translate transport only; Stage Interface owns the command contract and descriptors.

Benefits

  • leverage: one host contract
  • locality: host variance
  • new adapters stay small
